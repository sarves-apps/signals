Api calls - Using Observable

{{observableString}}

Type: todos$ is of type Observable.

Behavior: It represents an observable stream of data that can emit multiple values over time.

Usage: You would typically subscribe to this observable in your component to get the data and handle it (e.g., displaying it in the template). The $ suffix is a common naming convention to indicate that the variable is an observable.

{{signaleString}}

Type: todos is not an observable but a reactive signal.

Behavior: toSignal is a function that converts the observable returned by getTodos() into a reactive signal with an initial value of [].

Usage: Signals are part of Angular's reactive system, which allows you to react to changes in data and automatically update the template without manually subscribing. This can make your code cleaner and easier to manage.

@for (todo of todos(); track todo.id) {
{{ todo.title }}
}
